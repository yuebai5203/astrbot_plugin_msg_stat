📊 AI 消息 & Token 统计

AstrBot 插件 — 统计 AI 发言次数与 Token 用量，持久化追踪，每日报告。

AstrBot ≥ 4.0 Apache 2.0 QQ / NapCat

✨ 功能

📝 消息计数

自动记录 AI 在群聊和私聊中发出的每条消息

📊 合并转发

看看消息 — 当前群聊统计表格 + 合并转发最近消息

🌐 全局报表

看看全部消息 — 所有群聊和私聊发言数，含群名/昵称，发送到指定群

🔢 Token 追踪

实时追踪每次 LLM 调用的输入/输出 Token，看看token 随时查看

💾 持久化 + 每日报告

累计 Token 保存到文件重启不丢。每天 00:00 自动发送昨日统计到汇报群

📋 指令

指令说明
看看消息当前群聊 1h/30min 统计 + 合并转发
看看全部消息全局报表，发到配置的汇报群
看看token今日和累计 Token 用量

⚠️ 所有指令仅 AstrBot 管理员可用，群聊需 @bot。

⚙️ 配置

配置项说明默认
机器人显示名称合并转发中显示的昵称留空=QQ号
汇报群号报表投递目标群（QQ 群号）必填
启用每日报告每天 00:00 自动发报告✅
启用 Token 追踪追踪 Token 用量✅
合并转发最大条数看看消息转发条数上限90
转发消息截断长度单条消息截断字符数500

📦 安装

  1. AstrBot 后台 → 插件市场 → 搜索「AI消息&Token统计」→ 安装
  2. 或手动：将本仓库放入 data/plugins/ 目录
  3. 在插件配置面板设置 汇报群号
  4. 重载插件即可使用
